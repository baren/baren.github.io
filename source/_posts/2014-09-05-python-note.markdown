* 实际上，每一个python函数都有返回值，如果有return，则返回return指定的值；没有就返回None。

* python的搜索路径是在sys.path中搜索，sys.path是一个list，可以修改这个list的值。

* python中，所有都是对象（函数是对象，模块是对象等），意思是所有都可以赋值给变量，或者作为参数传递给函数。所有都是对象，有些可能不具有方法和属性，有些不能被继承，但是都可以赋值和传参。
